# K-means クラスタリング実装

## 概要

k-meansアルゴリズムを用いたクラスタリングの実装と可視化を行う．

## k-meansアルゴリズムについて

### アルゴリズムの流れ

1. ランダムに初期クラスタの中心を選択
2. 各データポイントを最も近いクラスタ中心に割り当て
3. 各クラスタの中心を再計算(そのクラスタに属するデータの平均値)
4. 収束するまで繰り返す

## 必要なライブラリ

pip install numpy matplotlib scikit-learn

- 今回のデータでは、4〜6個のクラスタが適切であると考えられる
